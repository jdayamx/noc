Network Operations Center (NOC)

The NOC Dashboard is a web-based application designed for monitoring system performance and network activity. It provides an overview of critical system resources, including CPU load, memory usage, disk space, and network interfaces. Users can track running processes, observe real-time network connections, and analyze ARP tables for active devices on the network.

The dashboard helps system administrators identify performance bottlenecks, monitor resource consumption, and analyze network activity in a centralized interface. With a focus on efficiency and simplicity, it provides key insights into system health while ensuring secure access through user authentication.
